# 🪨✂️📄 Rock, Paper, Scissors by KET_KAT

Классическая консольная игра "Камень, ножницы, бумага", написанная на Python. Игрок соревнуется против искусственного интеллекта компьютера.

## 🚀 Особенности проекта
* **Честный ИИ**: Компьютер делает свой выбор абсолютно случайно с помощью модуля `random`.
* **Ведение счета**: Игра запоминает текущий счет матча между игроком и компьютером.
* **Проверка ввода**: Защита от неверного ввода со стороны пользователя.
* **Фирменный стиль**: Фирменное приветствие от команды разработчиков `KET_KAT`.

## 🛠 Технологии проекта
* Python 3.12.9
* Встроенные модули: `random`, `os`, `time`

## 📦 Запуск игры
Проект не требует установки сторонних библиотек. Просто скачайте файл `main.py` и запустите его:
```bash
python main.py
```
